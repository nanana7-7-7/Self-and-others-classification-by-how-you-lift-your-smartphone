# Activity Recognition using Smartphone Sensors

## 概要

スマートフォンの加速度センサデータを用いて、ユーザーの動作特徴を分類する機械学習プロジェクトです。
持ち上げ方や動作パターンから個人差を抽出し、「本人らしさ」を利用した軽量認証や行動認識への応用を目的としています。

## 背景

スマートフォンには加速度センサやジャイロセンサなど多くのセンサが搭載されています。
本プロジェクトでは、それらのセンサデータに含まれる人ごとの特徴に着目し、自然な動作だけでユーザー判定や行動分類ができないかを検証しました。

## 以下のモデルを用いて分類を実施しました。

Random Forest
SVM
KNN
Decision Tree
MLPClassifier

## 特徴量

センサデータから以下の特徴量を抽出しています。

Mean
Standard Deviation
Max / Min
MAD（Median Absolute Deviation）

時系列データを統計量へ変換することで、動作パターンを分類可能にしました。

## 可視化

取得した x・y・z 軸の加速度データをグラフ化し、
移動平均を用いた波形の変化も確認しています。
